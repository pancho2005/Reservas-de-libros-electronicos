# Sistema de Gestión de Libros Electrónicos

## Descripción del Proyecto
El Sistema de Gestión de Libros Electrónicos es una aplicación diseñada para administrar una biblioteca digital, permitiendo la gestión de usuarios, libros electrónicos, reservas y la generación de reportes básicos. El sistema está orientado a organizar y controlar el acceso a los recursos digitales de manera eficiente.

Este proyecto corresponde a la **Etapa 1: Planeación del Software**, donde se define la estructura, los módulos y el alcance del sistema.

---

## Objetivo del Sistema
Desarrollar la planeación de un sistema que permita:
- Administrar libros electrónicos
- Gestionar usuarios
- Controlar reservas de libros
- Generar reportes básicos de uso

Todo esto aplicando un enfoque de **programación funcional**.

---

## Módulos del Sistema

### Módulo de Usuarios
- Registro de usuarios
- Almacenamiento de nombre y apellido
- Identificación de usuarios para la realización de reservas

---

### Módulo de Gestión de Libros
- Registro de libros electrónicos
- Información del libro (nombre, autor, número de páginas, año)
- Control de stock
- Consulta y actualización de libros

---

### Módulo de Gestión de Reservas
- Registro de reservas de libros
- Asociación entre usuario y libro
- Registro de fecha de reserva
- Registro de fecha de entrega
- Validación de disponibilidad según el stock

---

###Módulo de Reportes
- Libros más reservados
- Reporte de usuarios activos
- Registro de reservas realizadas
- Análisis básico del uso del sistema

---

## Alcance del Proyecto
El sistema incluye:
- Gestión de usuarios
- Gestión de libros electrónicos
- Control de reservas
- Generación de reportes básicos

No incluye:
- Pagos
- Multas por retraso
- Notificaciones automáticas
- Integraciones externas
- Sistemas avanzados de derechos digitales (DRM)

---

## Enfoque de Programación
El sistema se plantea bajo un enfoque de **programación funcional**, caracterizado por:
- Uso de funciones puras
- Separación entre datos y lógica
- Ausencia de estados globales
- Modularidad y reutilización de funciones

---

## Tecnologías (Planeación)
- Lenguaje de programación: go
- Control de versiones: Git y GitHub
